HomeAR is a Real Estate touring app that allows the user to do home tours from wherever there is an internet connection. It allows the user to view the interior of the house and interact with the utilities and furniture inside of the virtual home. The application would allow realtors to show their clients more potential houses and zero in on potential candidates instead of visiting multiple houses.
